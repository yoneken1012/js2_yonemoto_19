# ①課題名
DRAGON BALL FIGHTERZ じゃんけんバトル（Firebase Realtime DB 連携版）

## ②課題内容（どんな作品か）
- ドラゴンボールの世界観をモチーフにした、キャラクター選択型のじゃんけんバトルゲームです
- 勝敗ログはリアルタイム保存・取得され、画面下部へチャット形式で表示されます

## ③アプリのデプロイURL
- https://github.com/yoneken1012/js2_yonemoto_19

## ④アプリのログイン用IDまたはPassword（ある場合）
- ID: 今回なし
- PW: 今回なし

## ⑤工夫した点・こだわった点
- Firebase Realtime Database を利用し、勝敗ログをリアルタイムに保存＆取得
- 取得したログは画面下部に表示
- 各メッセージには 削除ボタン を実装し、押すと Firebase 上のデータもリアルタイムで削除される

## ⑥難しかった点・次回トライしたいこと（又は機能）
- Firebaseの接続エラー調査：パス誤り、push のタイミングなど、
書き込みが null になる複数要因の切り分けに苦戦
- 既存バトルゲームとの組み合わせ：キャラクター選択・HPバトル・演出など既存機能が多いため、ログ記録のタイミングと整合性が崩れないように組み込むのが難しかった
- 次回トライしたいこと：①ログイン機能（Firebase Auth）、②プレイヤーごとの戦績保存・勝率表示
